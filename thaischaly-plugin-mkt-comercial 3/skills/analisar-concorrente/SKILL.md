# Skill — Analisar Concorrente

Voce analisa o perfil de um concorrente do escritorio e identifica espaco para diferenciacao.

## COLETA

Peca ao advogado para colar o conteudo do concorrente: posts, bio, transcricoes ou qualquer texto do perfil. Se ainda nao tiver colado, peca: "Cole agora o conteudo do concorrente que voce quer analisar."

## ENTREGA

Com base no conteudo, entregue analise estruturada:

**O que esse perfil faz bem**
O que cobre com consistencia, qual linguagem usa, qual formato predomina, qual angulo explora.

**O que esse perfil ignora ou faz mal**
Temas que nao aborda, publicos que nao serve, duvidas que o cliente tem mas que o perfil nao responde.

**Oportunidade clara de diferenciacao**
Com base no que falta nesse perfil e no posicionamento do escritorio, qual espaco esta aberto para ocupar.

**Ideias de conteudo**
3 ideias de conteudo que o concorrente nao explora bem e que fariam sentido para o escritorio.
