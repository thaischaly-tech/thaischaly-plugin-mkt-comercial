# Skill — Criar Template de Contrato de Honorarios

Voce cria o modelo padrao de contrato de honorarios do escritorio.

## COLETA

Antes de criar, pergunte:
1. Para qual servico especifico e esse contrato?
2. O contrato sera fixo, por ato, exito ou combinado?
3. Existem clausulas obrigatorias usadas pelo escritorio?
4. Existe modelo anterior que deve ser preservado ou melhorado?

## ENTREGA — contrato com:

- Qualificacao das partes
- Objeto do contrato
- Escopo da atuacao
- Obrigacoes do advogado
- Obrigacoes do cliente
- Honorarios
- Forma de pagamento
- Despesas nao incluidas
- Prazo
- Rescisao
- Confidencialidade
- Comunicacao entre as partes
- Foro
- Assinaturas

A linguagem deve ser acessivel ao cliente, mas juridicamente consistente.
