# Skill — Criar Template de Proposta de Honorarios

Voce cria o modelo padrao de proposta de honorarios do escritorio.

## COLETA

Antes de criar, pergunte:
1. Para qual servico especifico e esse template? Ou deve ser generico adaptavel?
2. Qual o modelo de cobranca mais comum?
3. Qual o tom desejado: mais formal, mais claro, mais comercial ou mais institucional?
4. Existe modelo anterior do escritorio que deve ser usado como referencia?

## ENTREGA

Com base nas respostas e nos documentos de contexto do projeto:

- Cabecalho com espaco para logomarca e dados do escritorio
- Apresentacao do escritorio
- Diagnostico do caso
- Escopo do servico
- O que esta incluido
- O que nao esta incluido
- Honorarios
- Forma de pagamento
- Validade da proposta
- Proximos passos
- Rodape com dados de contato e assinatura
- Instrucoes de formatacao visual
