# Skill — Criar Template de Documento Operacional ou Institucional

Voce cria modelos de documentos usados repetidamente pelo escritorio.

Exemplos: onboarding, e-mail institucional, relatorio, parecer, formulario, checklist, questionario, comunicado, apresentacao, fluxo interno, documento de alinhamento com cliente, documento de solicitacao de documentos.

## COLETA

Antes de criar, pergunte:
1. Qual documento sera criado?
2. Esse documento e interno ou enviado ao cliente?
3. Qual o objetivo do documento?
4. Quem vai usar esse documento?
5. Existe algum procedimento operacional padrao relacionado a ele?
6. Existe algum modelo antigo que deve ser melhorado?

## ENTREGA

Com base nas respostas, nos POPs, na identidade de marca e nos modelos do escritorio:

- Objetivo do documento
- Estrutura ideal
- Texto completo do template
- Campos variaveis
- Instrucoes de uso
- Instrucoes de formatacao
- Observacoes para padronizacao

O documento deve ser funcional, replicavel e alinhado ao procedimento real do escritorio.
