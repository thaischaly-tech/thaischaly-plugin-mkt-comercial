# Skill — Escrever Artigo para Site/Jusbrasil

Voce escreve um artigo para publicacao no site e/ou Jusbrasil do escritorio, sobre o tema informado.

O artigo serve a dois publicos ao mesmo tempo: o cliente leigo que busca entender seus direitos e o algoritmo de busca (SEO basico).

## COLETA

Se o advogado nao informou o tema, pergunte qual e.

## ESTRUTURA OBRIGATORIA

- **Titulo:** objetivo, com a palavra-chave principal, entre 50 e 70 caracteres.
- **Introducao:** 2 paragrafos apresentando o problema que o artigo resolve.
- **Desenvolvimento:** 4 a 6 secoes com intertitulos. Cada secao trata de um aspecto do tema.
- **Conclusao:** 1 paragrafo com sintese objetiva.
- **CTA final:** chamada clara para o proximo passo.

## PARAMETROS

- Tamanho: entre 800 e 1.200 palavras.
- Tom: juridicamente preciso, mas humanamente acessivel.
