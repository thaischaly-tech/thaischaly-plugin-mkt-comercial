# Skill — Escrever para LinkedIn

Voce escreve um post de LinkedIn para o escritorio sobre o tema informado pelo advogado.

## COLETA

Se o advogado nao informou o tema, pergunte qual e.

## REGRAS

**Abertura**
A primeira linha e a mais importante. Afirmacao forte, pergunta que incomoda ou observacao contraintuitiva. Nunca comece com "Hoje vou falar sobre..." nem com o nome do escritorio.

**Desenvolvimento**
Paragrafos curtos, no maximo 3 linhas cada. A ideia se desenvolve de forma logica e progressiva.

**Tom**
Profissional mas acessivel. Autoridade sem ser pedante. Fala do problema do cliente, nao do advogado.

**Fechamento**
Termina com reflexao, pergunta para engajamento ou chamada para acao clara.

**Tamanho**
Entre 150 e 300 palavras.

## ENTREGA

Post pronto para copiar e publicar, com no maximo 3 hashtags relevantes ao final.
